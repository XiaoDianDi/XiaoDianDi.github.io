2. 安装过程中弹出对话框

目前来看，在安装过程中只弹出来一种对话框。

对话框提示内容：

  • 如果你将IDM安装到所选的目录，IDM将不能在保护模式下的IE内集成。

    如果需要集成到保护模式下的IE，必须安装到C盘。

    你想将目录切换到C盘么？

    是（切换到C盘） 否（保持安装到其他盘）

出现的原因：
  • 更换了IDM的默认安装目录。
解决方案：
1. 点击是，将安装目录切换回C盘。
2. 点击否，继续安装到其他盘。

3. 如何更改语言为中文

正常情况下，在中文的Windows上IDM在安装完成之后会提示需要下载中文语言包，下载安装完成，重新打开IDM即可。

不正常的情况下
  1. 下载IDM官方的语言包：链接
  2. 解压语言包到安装目录的Languages文件夹内。例如：C:\Program Files (x86)\Internet Download Manager\Languages

  3. 重新启动IDM在View->Language内选择->简体中文

4. 如何关联Firefox，chrome以及国产双核浏览器

1. 如何安装浏览器扩展

  • 1.1 chrome及国内的双核浏览器如何添加(火狐直接看最后的，IE请查看其他教程)。 (1)打开浏览器的扩展工具页面，不要关闭。 (2)打开C(D/E/F...):Program Files (x86)\Internet Download Manager 寻找IDMGCExt.crx (3)单击按住IDMGCExt.crx文件不要放，拖动到2.1步打开的扩展工具页面，然后放开，提示是否安装，安装。重启浏览器OK。

  • 1.2 firefox(火狐/FF)浏览器如何添加 (1)打开firefox (2)打开C(D/E/F...):Program Files (x86)\Internet Download Manager 寻找idmmzcc.xpi (3)单击按住idmmzcc.xpi文件不要放，然后拖放到火狐中，放开，提示安装，安装。重启浏览器OK。

2. 如何关联其他chromium（webkit/极速）内核浏览器

  • 2.1 右击浏览器快捷方式，点击属性，点击打开文件位置，不要关闭弹出的窗口

    以chrome为例

  • 2.2 打开IDM，点击下载->选项->添加浏览器，从弹出的窗口中选择第一步中找到的可执行文件（exe文件），选中即可

    同样以chrome为例

  • 2.3 安装IDM的扩展工具，见1.1所述.

6. 如何破解使用（推荐在力所能及范围内支持正版，让好的软件走的更远）

有两种简单方式进行破解。

第一种
  1. 进入IDM吧群共享：链接
  2. 选择自己喜欢的破解版本
  3. 安装下载好的破解版本
第二种
  1. 进入IDM吧群共享：链接
  2. 在官方原版文件夹内，下载最新的官方安装包
  3. 在patch文件夹内下载 看雪破解文件
  4. 安装原版安装包，并关闭IDM
  5. 将看雪破解文件内的 dll 解压到 IDM的安装目录（压缩包内含README文档，有操作方法。）
  6. 重新打开IDM

7. IDM可以做到哪些，不能做哪些

1. IDM可以做哪些？
  • IDM可以下载普通的HTTP，FTP链接
  • IDM可以抓取某些在线视频（例如YouTube）
2. IDM不能做哪些？
  • IDM不能突破你的带宽极限，它只能不留余力的去下载
  • IDM不能下载某些加密的在线视频（大多数的教程网站）
  • IDM不能下载 磁力链接，迅雷链接，种子文件，电驴链接，旋风链接。（仅仅支持 http/https/ftp 开头的下载链接）
